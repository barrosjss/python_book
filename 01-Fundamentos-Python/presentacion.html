¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, diseñado para que el código sea fácil de leer y escribir.

Alto nivel

Escribe instrucciones cercanas al lenguaje humano, no al lenguaje de la máquina.

Interpretado

No necesita compilarse. El código se ejecuta línea a línea en tiempo real.

Propósito general

Sirve para web, datos, IA, scripts, ciencia, juegos y mucho más.

Historia de Python

1989

Guido van Rossum comienza a desarrollarlo

Durante las vacaciones de Navidad en el Centrum Wiskunde & Informatica (CWI), Países Bajos. Lo llamó "Python" por el show de comedia Monty Python's Flying Circus.

1991

Python 0.9.0 — Primera versión pública

Ya incluía clases, manejo de excepciones, funciones y módulos. Una base sólida desde el inicio.

2000

Python 2.0 — Comprensión de listas y recolector de basura

Gran salto en productividad. Fue la versión dominante por más de una década.

2008

Python 3.0 — Rediseño total del lenguaje

Incompatible con Python 2, pero más limpio y consistente. Hoy es el estándar.

Hoy

El lenguaje #1 del mundo

Lidera rankings globales (TIOBE, Stack Overflow, IEEE). Más de 8 millones de desarrolladores activos.

El Zen de Python

Python tiene una filosofía de diseño escrita por Tim Peters. Se puede ver ejecutando import this en cualquier terminal Python.

✦ Lo bello es mejor que lo feo.
✦ Lo explícito es mejor que lo implícito.
✦ Lo simple es mejor que lo complejo.
✦ La legibilidad importa.
✦ Los errores nunca deben pasar en silencio.
✦ Debe haber una — y preferiblemente solo una — manera obvia de hacerlo.
✦ Si es difícil de explicar, es una mala idea.
✦ Si es fácil de explicar, puede ser una buena idea.

Características de Python

  • Sintaxis limpia: El código se lee casi como pseudocódigo en inglés. Menos líneas para hacer lo mismo.
  • Tipado dinámico: No declaras el tipo de una variable. Python lo detecta automáticamente.
  • Multiparadigma: Soporta programación orientada a objetos, funcional y procedural.
  • Multiplataforma: El mismo código corre en Windows, macOS y Linux sin cambios.
  • Baterías incluidas: Viene con una librería estándar enorme. No necesitas instalar nada para hacer cosas básicas.
  • Ecosistema gigante: Más de 500,000 paquetes en PyPI (pip) para cualquier necesidad.
  • Gran comunidad: Documentación excelente, foros activos, respuesta rápida a problemas.
  • Open source: Gratis, siempre. Nadie te cobra por usarlo en producción.

Python vs otros lenguajes — Legibilidad

El mismo programa "Hola Mundo" en distintos lenguajes:

Java

public class Main {
  public static void main(String[] args) {
    System.out.println("Hola Mundo");
  }
}

C++

#include <iostream>
using namespace std;
int main() {
  cout << "Hola Mundo" << endl;
  return 0;
}

Python ✦

print("Hola Mundo")
Una sola línea. Sin imports, sin clases, sin llaves, sin punto y coma. Python elimina el ruido sintáctico para que te concentres en resolver el problema.

⚙️ Automatización y Scripting

Una de las razones por las que Python se popularizó tanto: permite automatizar tareas repetitivas con muy poco código.

  • Renombrar 1,000 archivos en segundos
  • Leer y procesar archivos Excel o CSV masivos
  • Enviar correos electrónicos automáticos
  • Hacer web scraping (extraer datos de sitios web)
  • Controlar el sistema operativo y archivos
  • Generar reportes PDF automáticos
  • Interactuar con APIs de servicios externos

Ejemplo: Renombrar 1,000 archivos

import os

carpeta = "mis_archivos"
archivos = os.listdir(carpeta)

for i, archivo in enumerate(archivos):
    nuevo_nombre = f"archivo_{i+1}.txt"
    os.rename(archivo, nuevo_nombre)

print("¡Listo! Archivos renombrados.")
Lo que tomaría horas haciendo manualmente, Python lo hace en milisegundos.

Instalación de Python

🪟 Windows

  1. Ir a python.org
  2. Descargar la versión más reciente
  3. Marcar "Add Python to PATH" (muy importante)
  4. Instalar → Verificar con python --version

🍎 macOS

  1. Opción 1: python.org (mismo proceso)
  2. Opción 2 (recomendada): brew install python3
  3. Verificar: python3 --version

🐧 Linux

Python 3 suele venir preinstalado. Verificar con python3 --version. Si no: sudo apt install python3

Verificar instalación en la terminal:

$ python --version
Python 3.12.0

$ python
Python 3.12.0 (main, Oct 2 2023)
>>> print("Hola")
Hola
>>>
⚠️ Windows: Si el comando python no funciona, cierra y abre de nuevo la terminal después de instalar. El PATH necesita recargarse.

Tu primer programa en Python

Por convención, el primer programa en cualquier lenguaje imprime "Hello, World!". En Python:

print("Hello, World!")

Un programa más interactivo:

# Pedimos el nombre al usuario
nombre = input("¿Cómo te llamas? ")

# Saludamos con su nombre
print(f"¡Hola, {nombre}! Bienvenido a Python 🐍")

# Mostramos información básica
edad = int(input("¿Cuántos años tienes? "))
print(f"En 10 años tendrás {edad + 10} años.")

Lo que aprendemos con este programa:

  • print() — muestra texto en pantalla
  • input() — lee texto que escribe el usuario
  • int() — convierte texto a número entero
  • f"..." — f-string: inserta variables dentro de texto
  • # — comentario: Python lo ignora al ejecutar
  • Variables se crean solo asignando: nombre = "valor"
Sin punto y coma. Sin llaves. Sin tipos. Python elimina todo lo que no agrega valor al código.

Estructura básica del código Python

Python usa indentación (espacios al inicio de línea) para definir bloques de código. No hay llaves {}.

# Variables — sin declarar tipo
nombre = "Ana"
edad   = 25
activo = True

# Condicional — la indentación delimita el bloque
if edad >= 18:
    print("Es mayor de edad")
else:
    print("Es menor de edad")

# Bucle
for i in range(3):
    print(f"Iteración {i}")

# Función
def saludar(nombre):
    return f"Hola, {nombre}!"
La indentación NO es opcional en Python. A diferencia de otros lenguajes donde es solo estilo, aquí determina qué código está dentro de un bloque.

Convención: usar 4 espacios por nivel de indentación (VS Code lo hace automáticamente).

⚠️ Error común: Mezclar tabs y espacios. Usar siempre uno de los dos. Python 3 lanza error si los mezclas.
Ventaja: Al obligar la indentación, Python garantiza que todo código bien escrito sea también fácil de leer.

pip — El gestor de paquetes de Python

Python viene con pip, una herramienta que te permite instalar cualquiera de los 500,000+ paquetes disponibles en internet con un solo comando.

Es como una "tienda de aplicaciones" para Python: encuentras una librería que resuelve tu problema, la instalas en segundos y ya está disponible en tu código.

Paquetes populares y para qué sirven:

requests — HTTP pandas — datos flask — web numpy — matemáticas pillow — imágenes selenium — automatización web sqlalchemy — bases de datos fastapi — APIs

Comandos esenciales de pip:

# Instalar un paquete
$ pip install requests

# Instalar versión específica
$ pip install requests==2.31.0

# Ver paquetes instalados
$ pip list

# Desinstalar
$ pip uninstall requests

# Guardar dependencias del proyecto
$ pip freeze > requirements.txt

# Instalar desde requirements.txt
$ pip install -r requirements.txt
