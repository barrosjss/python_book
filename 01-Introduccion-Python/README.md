# Bloque 1: Introducción a Python y Setup 🛠️

Bienvenido al primer paso de tu viaje en Python. Aquí prepararemos tu "estudio de arquitectura" para que puedas empezar a construir.

## 🎯 Objetivos de este módulo
1.  Instalar Python en tu computadora.
2.  Configurar Visual Studio Code como tu editor principal.
3.  Entender cómo ejecutar un script de Python.
4.  Tu primer comando: `print()`.

---

## 1. Instalación de Python

Python es un lenguaje interpretado, lo que significa que necesitas un "traductor" (el intérprete) instalado en tu sistema.

-   **Windows/Mac/Linux**: Descárgalo desde [python.org](https://www.python.org/downloads/).
-   **Verificación**: Abre una terminal (o CMD) y escribe:
    ```bash
    python --version
    ```

---

## 2. Configuración de VS Code

Visual Studio Code es el editor más popular hoy en día. Para que sea tu mejor aliado con Python, instala la extensión oficial:

1.  Abre VS Code.
2.  Ve al ícono de **Extensions** (o presiona `Ctrl+Shift+X`).
3.  Busca **"Python"** (la de Microsoft) e instálala.

---

## 3. Del Pseudocódigo a la Realidad

En tu clase de lógica aprendiste que para mostrar algo en pantalla usamos una instrucción de **SALIDA**.

**En Pseudocódigo:**
```text
ESCRIBIR "Hola Estudiantes"
```

**En Python:**
```python
print("Hola Estudiantes")
```

### 🔬 Hagamos la prueba
Crea un archivo llamado `hola_mundo.py` en esta carpeta y escribe el código de arriba. Luego ejecútalo desde VS Code con el botón de **Play** o desde la terminal:
```bash
python hola_mundo.py
```
