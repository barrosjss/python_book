Módulo 2 — Tipos de Datos

¿Qué son los datos primitivos?

Todo valor en Python tiene un tipo. Los datos primitivos son los tipos más básicos que existen — los que Python conoce desde el inicio, sin necesidad de importar nada.

🔤
str

🔢

print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type(True))        # <class 'bool'>
print(type("Hola mundo")) # <class 'str'>
print(type(None))        # <class 'NoneType'>

type() es una función nativa de Python que devuelve el tipo de cualquier valor. Útil para entender qué estás manejando.

Python detecta el tipo automáticamente según cómo escribes el valor. No necesitas declararlo.

🔤 String str

Un str (string) es una cadena de caracteres. Puede contener letras, números, símbolos y espacios. Siempre va entre comillas.

# Comillas simples o dobles — igual
nombre  = "Ana"
saludo  = 'Hola mundo'

# Comillas triples — texto multilínea
mensaje = """Este es
un mensaje
en varias líneas."""

# f-string — insertar variables en texto
edad = 25
print(f"Tengo {edad} años")

Concatenación — unir strings

a = "Hola"
b = "mundo"
print(a + " " + b)  # Hola mundo

Repetición

print("=" * 20)
# ====================
len() devuelve la longitud del string:
len("Python") → 6

⚠️ Strings son inmutables

No puedes cambiar caracteres individuales. Debes crear un string nuevo.

🔢 Integer int

Números enteros, positivos o negativos, sin parte decimal. En Python no hay límite de tamaño para un entero.

a = 10              # positivo
b = -5             # negativo
c = 0               # cero
d = 123456789012345 # muy grande: sin problema
⚠️ Python NO tiene ++ ni --
Esos operadores son de JavaScript/C.
En Python se usa: x += 1 y x -= 1

📐 Float float

Números con parte decimal. Python los almacena como punto flotante de 64 bits (doble precisión).

pi      = 3.14159
precio  = 9.99
negativo= -0.5
grande  = 1.5e10   # notación científica = 15,000,000,000

# División SIEMPRE produce float
print(10 / 2)    # 5.0 — no 5
print(10 // 2)   # 5  — división entera
⚠️ Precisión de float
0.1 + 0.2 → 0.30000000000000004
Limitación del hardware, no de Python. Para dinero usar decimal.Decimal.

Conversión entre tipos numéricos:

x = 5
print(float(x))   # 5.0 — int a float

y = 3.9
print(int(y))     # 3   — trunca, no redondea

print(round(3.9)) # 4   — redondear
print(round(3.14159, 2)) # 3.14

Funciones útiles con números

  • abs(-5) → 5 (valor absoluto)
  • max(3, 7, 1) → 7
  • min(3, 7, 1) → 1
  • sum([1,2,3]) → 6

✅ Boolean bool

Solo dos valores posibles: True o False. La base de toda lógica condicional. Con mayúscula inicial en Python.

activo   = True
bloqueado= False

# Cualquier valor puede convertirse a bool
print(bool(0))        # False
print(bool(1))        # True
print(bool(""))       # False — string vacío
print(bool("hola"))   # True
print(bool([]))       # False — lista vacía
print(bool([1,2]))    # True

∅ NoneType None

None representa la ausencia de valor. Es el equivalente a null en otros lenguajes. No es 0, no es "", no es False — es literalmente la nada.

resultado = None

print(resultado)          # None
print(type(resultado))    # <class 'NoneType'>

# Comparar con None → usar "is", no "=="
print(resultado is None)  # True ✓ forma correcta
print(resultado == None) # True  (funciona pero no recomendado)

¿Cuándo aparece None en la práctica?

  • Una función que no retorna nada devuelve None implícitamente.
  • Variable que todavía no tiene valor asignado.
  • Resultado de búsqueda que no encontró nada.
def sin_retorno():
    print("Hola")
    # No hay return

x = sin_retorno()
print(x)   # None

🔄 Conversión de tipos (Type Casting)

Python permite convertir valores de un tipo a otro usando funciones nativas.

# str → int
edad_texto = "25"
edad       = int(edad_texto)   # 25

# int → str
numero     = 42
texto      = str(numero)       # "42"

# str → float
precio_txt = "9.99"
precio     = float(precio_txt) # 9.99

# input() SIEMPRE devuelve str
edad_raw   = input("Tu edad: ")
edad       = int(edad_raw)      # convertir antes de operar
