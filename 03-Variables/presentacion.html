Módulo 3 — Variables & Strings

¿Qué es una variable?

Una variable es un contenedor con nombre que almacena un valor en memoria. En lugar de repetir el mismo valor en todo el código, le das un nombre y lo reutilizas.

Analogía: una variable es como una caja etiquetada. La etiqueta es el nombre, el contenido es el valor. Puedes abrir la caja, ver qué hay dentro, o cambiar el contenido. 
# Asignar un valor a una variable
nombre = "Carlos"     # str
edad   = 30           # int
activo = True         # bool
precio = 15000        # int

# Usar la variable en cualquier parte del código
print(f"Hola {nombre}, tienes {edad} años")

Variables guardan cualquier tipo de dato:

texto        = "Hola"       # str
numero       = 42           # int
decimal      = 3.14         # float
activo       = True         # bool
lista        = [1, 2, 3]    # list
diccionario  = {"a": 1}    # dict
Python detecta el tipo automáticamente según el valor que asignas. No necesitas declarar int x = 5 como en Java o C.

Reasignación y asignación múltiple

En Python puedes cambiar el valor de una variable en cualquier momento. Incluso puedes cambiarle el tipo.

# Reasignación — el valor cambia
contador = 0
print(contador)   # 0

contador = 10
print(contador)   # 10

contador = contador + 1
print(contador)   # 11

# Forma corta con operadores de asignación
contador += 5    # contador = contador + 5 → 16
contador *= 2    # contador = contador * 2 → 32

Asignación múltiple — varias variables en una línea:

# Asignar el mismo valor a varias variables
a = b = c = 0

# Asignar distintos valores en una línea
x, y, z = 1, 2, 3
print(x, y, z)   # 1 2 3

# Intercambiar valores sin variable temporal
a, b = 10, 20
a, b = b, a
print(a, b)      # 20 10
El intercambio a, b = b, a es Python puro — en otros lenguajes necesitas una variable temporal.

📥 input() — Variables desde el usuario

input() pausa el programa y espera que el usuario escriba algo. Lo que escribe queda guardado en una variable.

1

Muestra un mensaje

El texto dentro de los paréntesis aparece en pantalla como pregunta.

2

Espera al usuario

El programa se detiene hasta que el usuario escribe y presiona Enter.

3

Devuelve siempre str

Sin importar qué escriba el usuario — siempre es texto. Debes convertirlo si necesitas otro tipo.

# Básico
variable = input("Mensaje para el usuario: ")

# En la práctica
nombre = input("¿Cómo te llamas? ")
edad   = int(input("¿Cuántos años tienes? "))
precio = float(input("¿Cuánto cuesta? "))

print(f"Hola {nombre}, tienes {edad} años")
⚠️ Error frecuente:
edad = input("Edad: ")
print(edad + 1) → TypeError
input() devuelve str. No puedes sumar str con int sin convertir.

f-strings — Formateo moderno

Un f-string es un string con la letra f antes de las comillas. Dentro puedes insertar cualquier expresión Python entre { }.

nombre = "Ana"
edad   = 25

# f-string — forma recomendada desde Python 3.6+
print(f"Hola, mi nombre es {nombre} y tengo {edad} años.")

# Puedes poner expresiones directamente
print(f"El doble de tu edad: {edad * 2}")
print(f"Nombre en mayúsculas: {nombre.upper()}")
print(f"2 + 2 = {2 + 2}")
Disponible desde Python 3.6. Es la forma más legible, rápida y recomendada hoy. Úsala siempre.

Formas de formatear strings en Python (contexto histórico):

f-string — Python 3.6+ ✓ Usar

print(f"Hola {nombre}")

.format() — Versión anterior

print("Hola {}".format(nombre))

% — Versión antigua

print("Hola %s" % nombre)

f-strings — Formateo avanzado

Dentro de { } puedes controlar cómo se muestra el valor con :{especificador}

pi      = 3.14159265
numero  = 1234567.89
nombre  = "Ana"

# Limitar decimales
print(f"Pi: {pi:.2f}")        # Pi: 3.14
print(f"Pi: {pi:.4f}")        # Pi: 3.1416

# Número con separador de miles
print(f"{numero:,.2f}")       # 1,234,567.89

# Rellenar con ceros
print(f"{42:05d}")           # 00042
# Alinear texto en columnas
print(f"|{'Hola':<10}|")       # |Hola      |  izquierda
print(f"|{'Mundo':>10}|")      # |     Mundo|  derecha
print(f"|{'Centro':^10}|")     # |  Centro  |  centrado

# Porcentajes
tasa = 0.1975
print(f"Tasa: {tasa:.1%}")      # Tasa: 19.8%

🔭 Alcance de variables (Scope)

El scope determina desde dónde se puede acceder a una variable. En Python hay dos ámbitos principales.

Variable Global

Definida fuera de cualquier función. Accesible desde cualquier parte del programa.

mensaje = "Hola"   # global

def saludar():
    print(mensaje)  # ✓ accede

saludar()           # Hola

Variable Local

Definida dentro de una función. Solo existe dentro de esa función.

def calcular():
    resultado = 42   # local
    print(resultado) # ✓

calcular()
print(resultado)      # ✗ NameError
Regla general: Define variables en el scope más pequeño que las necesites. Variables globales en exceso hacen el código difícil de entender.
