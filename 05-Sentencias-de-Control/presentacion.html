Módulo 05

Sentencias de Control

if / elif / else  ·  for  ·  while  ·  break  ·  continue

Aprende a tomar decisiones y repetir acciones en tus programas Python.

¿Qué son las sentencias de control?

Permiten alterar el flujo de ejecución del programa: decidir qué código se ejecuta y cuántas veces.

Sentencia Tipo ¿Cuándo usarla?
if / elif / else Condicional Cuando una acción depende de una condición
for Bucle Cuando se conoce el número de iteraciones
while Bucle Cuando se repite mientras una condición sea verdadera
break Control Salir del bucle antes de terminar
continue Control Saltar una iteración y continuar
💡 Todos los bloques de código en Python se definen con indentación (4 espacios). No hay llaves {}.
Sección 01

Condicionales

if · elif · else · operador ternario

Condicionales — if y else

if ejecuta un bloque solo si la condición es True. else captura el caso contrario.

Solo if

pythonif condición:
    # código si es True

if + else

pythonif condición:
    # si es True
else:
    # si es False

Ejemplo real:

pythonedad = int(input("Tu edad: "))

if edad >= 18:
    print("Puedes votar")
else:
    print("Aún no puedes votar")
✅ La indentación (4 espacios) define qué código pertenece al bloque. Sin ella, Python lanza error.

Condicionales — elif

elif (else if) verifica condiciones alternativas. Solo se ejecuta si las anteriores fueron False. 

pythonif condición1:
    # si condición1 es True
elif condición2:
    # si condición2 es True
elif condición3:
    # si condición3 es True
else:
    # ninguna se cumplió

Clasificación de notas:

pythonnota = int(input("Nota (0-100): "))

if nota >= 90:
    print("Excelente 🏆")
elif nota >= 70:
    print("Aprobado ✅")
else:
    print("Reprobado ❌")

Varios if vs if/elif/else

Diferencia clave: con varios if se evalúan todas las condiciones. Con if/elif/else se evalúa solo un bloque.

⚠️ Varios if — todas las condiciones se evalúan 
pythonx = 15

if x > 10:
    print("Mayor que 10")   # ✓
if x > 0:
    print("Positivo")       # ✓ también
# Se imprimen LOS DOS
✅ if/elif/else — se evalúa solo un bloque 
pythonx = 15

if x > 10:
    print("Mayor que 10")   # ✓
elif x > 0:
    print("Positivo")   # No se evalúa
else:
    print("Cero o negativo")
# Solo imprime "Mayor que 10"
📌 Usa if/elif/else cuando las condiciones son mutuamente excluyentes. Usa varios if cuando pueden cumplirse varias a la vez.

Operador Ternario

Forma compacta de escribir un if...else en una sola línea.

resultado = valor_si_true if condición else valor_si_false

Forma clásica:

pythonedad = 20
if edad >= 18:
    msg = "Mayor de edad"
else:
    msg = "Menor de edad"
print(msg)

Con ternario:

pythonedad = 20
msg = "Mayor de edad" if edad >= 18 \
      else "Menor de edad"
print(msg)

Directamente en print:

pythonprint("Acceso permitido" if edad >= 18 else "Acceso denegado")
⚠️ Útil para casos simples. No lo uses con lógica compleja — prefiere el if clásico para mantener legibilidad.
Sección 02

Bucles y Ciclos

for · range() · while · break · continue · else

Bucle for

Recorre un iterable (lista, string, rango…) ejecutando el bloque en cada elemento. Número de iteraciones definido. 

pythonfor variable in secuencia:
    # bloque a repetir

Con range():

pythonfor i in range(5):          # 0,1,2,3,4
    print(i)

for i in range(1, 6):       # 1,2,3,4,5
    print(i)

for i in range(0, 10, 2):   # 0,2,4,6,8
    print(i)

for — Recorriendo estructuras

Lista

pythonfrutas = ["mango", "pera", "uva"]
for fruta in frutas:
    print(fruta)

String (carácter a carácter)

pythonfor letra in "Python":
    print(letra)

Diccionario (claves y valores)

pythonnotas = {"Ana": 4.5, "Luis": 3.8}

for nombre, nota in notas.items():
    print(f"{nombre}: {nota}")

enumerate() — índice + valor

pythoncolores = ["rojo", "verde", "azul"]
for i, color in enumerate(colores):
    print(f"{i}: {color}")
💡 enumerate() entrega el índice y el valor juntos — no necesitas llevar un contador manualmente.

Bucle while

Repite el bloque mientras la condición sea True. Número de iteraciones indefinido. 

pythonwhile condición:
    # bloque a repetir

Ejemplo — contar hasta 5:

pythoncontador = 1
while contador <= 5:
    print(contador)
    contador += 1
🚨
Loop infinito: si la condición nunca se vuelve False, el programa se cuelga.
while True: → requiere break para salir.

Bucles — break y continue

🚫

break

Sale completamente del bucle, aunque la condición siga siendo True.

pythonfor numero in range(10):
    if numero == 5:
        break       # para en 5
    print(numero)
# Imprime: 0 1 2 3 4
⏭️

continue

Salta el resto de la iteración actual y va a la siguiente.

pythonfor numero in range(5):
    if numero == 2:
        continue    # salta el 2
    print(numero)
# Imprime: 0 1 3 4
🧭 break = "No pasarás" — corta el bucle.   continue = "Skip" — salta esa vuelta y sigue.

break y continue — Ejemplo completo

Un programa que pide números: ignora el 0 con continue y para con número negativo usando break.

pythonwhile True:
    numero = int(input("Ingresa un número: "))

    if numero == 0:
        print("Cero ignorado → continue\n")
        continue       # salta el print final, vuelve al while

    if numero < 0:
        print("Número negativo → deteniendo con break.")
        break          # sale completamente del bucle

    print(f"Número aceptado: {numero}\n")
📌 while True crea un bucle infinito intencional. Siempre debe tener un break para salir.

else en bucles

El bloque else de un bucle se ejecuta solo si el ciclo terminó normalmente (sin break).

Con for:

pythonfor n in [1, 3, 7]:
    if n == 5:
        break
else:
    print("No se encontró el 5")
# → imprime el mensaje (no hubo break)

Con while:

pythonintentos = 0
while intentos < 3:
    pwd = input("Contraseña: ")
    if pwd == "python":
        print("Acceso concedido")
        break
    intentos += 1
else:
    print("Cuenta bloqueada")
# else solo corre si agotó los 3 intentos
💡 El else en bucles es exclusivo de Python. Úsalo para detectar si un bucle terminó "exitosamente" sin interrupción.

List Comprehension — for en una línea

Forma compacta de crear listas usando for (y opcionalmente if) en una sola expresión. 

nueva_lista = [expresión for item in iterable if condición]

Forma clásica:

pythoncuadrados = []
for n in range(5):
    cuadrados.append(n ** 2)
# [0, 1, 4, 9, 16]

Con comprehension:

pythoncuadrados = [n ** 2 for n in range(5)]
# [0, 1, 4, 9, 16]

Con filtro if:

pythonpares = [n for n in range(10) if n % 2 == 0]   # [0, 2, 4, 6, 8]

mayusculas = [p.upper() for p in ["ana", "luis", "marta"]]   # ['ANA', 'LUIS', 'MARTA']

Actividad — Sistema de Calificaciones

Programa que pide notas de estudiantes, las clasifica y muestra estadísticas. Combina while + if/elif + for. 

pythonnotas = []
continuar = "s"

while continuar.lower() == "s":
    nombre = input("Nombre del estudiante: ")
    nota   = float(input("Nota (0-5): "))

    if nota >= 4.5:
        estado = "Excelente 🏆"
    elif nota >= 3.0:
        estado = "Aprobado ✅"
    else:
        estado = "Reprobado ❌"

    notas.append({"nombre": nombre, "nota": nota, "estado": estado})
    continuar = input("¿Agregar otro? (s/n): ")

print("\n--- RESULTADOS ---")
for e in notas:
    print(f"{e['nombre']:<20} {e['nota']:.1f}  {e['estado']}")

promedio = sum(e["nota"] for e in notas) / len(notas)
print(f"\nPromedio del grupo: {promedio:.2f}")
🎯 Reto: agrega un conteo de aprobados y reprobados al final.
