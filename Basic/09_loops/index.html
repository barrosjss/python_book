Estructuras de Control: Bucles (Iteraciones)

Dominando la repetición eficiente en algoritmos

1. Conceptos Teóricos

Un bucle es una estructura de control que permite ejecutar un bloque de código repetidamente mientras se cumpla una condición específica o hasta completar una secuencia.

  • Iteración: Cada una de las repeticiones del bucle.
  • Condición de parada: La expresión lógica que determina si el bucle continúa o finaliza (evita bucles infinitos).
  • Contador/Acumulador: Variables que suelen controlar el progreso del ciclo.

2. El Bucle while (Mientras)

Se utiliza cuando no sabemos de antemano cuántas veces se ejecutará el código, pues depende de una condición lógica.

Diagrama de Flujo: While

Ejemplo en Python

# Imprimir números del 1 al 5
contador = 1
while contador <= 5:
    print(f"Número: {contador}")
    contador += 1  # Incremento crucial

3. El Bucle for (Para)

Ideal para recorrer elementos en una colección (listas, rangos, cadenas) o cuando conocemos el número exacto de iteraciones.

Diagrama de Flujo: For

Ejemplo en Python

# Recorrer una lista de frutas
frutas = ["Manzana", "Pera", "Uva"]
for fruta in frutas:
    print(f"Hoy comeré: {fruta}")

# Usando un rango
for i in range(1, 6):
    print(f"Iteración número {i}")

4. Ejercicios Propuestos

Ejercicio 1: El Factorial

Escribe un programa que solicite un número entero positivo al usuario y calcule su factorial (n!) utilizando un bucle for o while.

Pista: 5! = 5 * 4 * 3 * 2 * 1 = 120.


Ejercicio 2: Validación de Entrada

Crea un programa que pida una contraseña al usuario. El programa debe seguir pidiendo la contraseña repetidamente hasta que el usuario ingrese la palabra clave "Python123".
