## 🛠️ ¿Cómo usar este repositorio?

Cada módulo sigue una estructura pensada para que no pierdas de vista la lógica:

1.  **Traducción de Sintaxis**: Cómo pasar del pseudocódigo que ya conoces a la sintaxis limpia de Python.
2.  **Ejemplos Comparativos**: Verás el diagrama de flujo (Mermaid) y su implementación inmediata en código.
3.  **Laboratorios Prácticos**: Retos donde deberás aplicar lo aprendido para resolver problemas reales.
4.  **De la Misión Crítica al Script**: Los mismos proyectos que analizaste en lógica, ahora funcionando en tu computadora.

## 🚀 Ruta de Aprendizaje

### 1️⃣ [Introducción y Setup](./01-Introduccion-Python/README.md)
Instalación de Python, configuración de VS Code y tu primer `print("Hola Mundo")`.

### 2️⃣ [Dominando los Datos](./02-Tipos-de-Datos/README.md)
Uso de `int`, `float`, `str` y `bool`. Entrada de datos con `input()` y salida con f-strings.

### 3️⃣ [Toma de Decisiones y Ciclos](./03-Estructuras-de-Control/README.md)
Implementación de `if`, `elif`, `else`. Automatización con `for` y `while`.

### 4️⃣ [El Poder de las Funciones](./04-Funciones/README.md)
Modularización, parámetros, retorno de valores y buenas prácticas (Docstrings).

### 5️⃣ [Colecciones y Listas](./05-Listas-y-Arreglos/README.md)
Manejo de múltiples datos, métodos de listas, slicing y recorridos eficientes.
