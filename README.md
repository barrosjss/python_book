## 🛠️ ¿Cómo usar este repositorio?

Cada módulo incluye una **presentación interactiva** (`presentacion.html` o `index.html`) y **actividades prácticas** en la carpeta `actividades/`.

1. **Abre la presentación** del módulo en tu navegador (doble clic o con Live Server en VS Code).
2. **Sigue las diapositivas** en clase: pseudocódigo, sintaxis Python, ejemplos y diagramas.
3. **Completa las actividades** creando tus propios archivos `.py` según las instrucciones.
4. **Conecta con lógica**: cada módulo retoma los conceptos que ya viste en diagramas de flujo y los traduce a código real.

## 🎯 Filosofía de Aprendizaje

1. **Diagrama primero**: Si no puedes dibujarlo, no intentes programarlo.
2. **Lógica universal**: Python es solo una herramienta; tu mente es el arquitecto.
3. **Código limpio**: Aprenderás no solo a que funcione, sino a que sea legible y profesional.
